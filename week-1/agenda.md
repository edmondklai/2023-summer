# Week 1

- Install brew on Mac
  - https://brew.sh/
  - `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- Install git
  - https://git-scm.com/download/mac
  - `brew install git`
- Install node
  - https://nodejs.org/en/download
- Install VSCode
  - https://code.visualstudio.com/
