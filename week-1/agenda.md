# Week 1

## Installation
- Install brew on Mac
  - https://brew.sh/
  - `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- Install git
  - https://git-scm.com/download/mac
  - `brew install git`
- Install VSCode
  - https://code.visualstudio.com/
