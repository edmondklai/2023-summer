# Week 1

## Installation
- Install Chrome
- Install brew on Mac
  - https://brew.sh/
  - `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- Open Github account
- Install git
  - https://git-scm.com/download/mac
  - `brew install git`
- Install VSCode
  - https://code.visualstudio.com/
